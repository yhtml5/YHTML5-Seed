### checklist

- [ ] 异常处理
  - [ ] 网络异常
  - [ ] 参数异常校验 [ajax传出, 传入, props]
  - [ ] promise 是否正确 catch
- [ ] 内存泄露
  - [ ] 全局变量
  - [ ] 遗漏的定时器
  - [ ] 循环引用 
- [ ] 代码规范
- [ ] react 
  - [ ] store 是否简介, 层级是否过深
  - [ ] 生命周期是否正确使用
  - [ ] render should only have display logic
  - [ ] 组件应该大写
- [ ] git commit message
